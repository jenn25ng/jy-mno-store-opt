# 매장수 최적화 서비스 기획서

SKT 오프라인 매장 포트폴리오 최적화 — 분석 방법론 및 서비스 설계

---

## 1. 문제 정의

### 배경
- SKT 오프라인 매장을 최적화해서 운영하는 방향으로 전략 전환 필요
- 단순 비용 절감이 아닌 **M/S 방어 + 수익성 개선 + 커버리지 유지**를 동시에 달성해야 함
- 의사결정을 데이터/AI 기반으로 전환하고자 함

### 핵심 질문

**"어떤 매장을 닫아도 SKT 점유율과 고객 경험에 타격이 없는가?"**

### 목적함수

매장 폐점 결정은 단일 지표로 환원 불가. 아래 3가지 렌즈를 동시에 평가함.

| 렌즈 | 핵심 질문 | 판단 기준 |
|---|---|---|
| **M/S 방어** | 닫아도 시장점유율이 유지되는가? | 인근 SKT 매장으로 가입 흡수 가능 여부 |
| **수익성** | 운영비 대비 창출 가치가 적자인가? | 비용 대비 LTV·순증 기여도 |
| **커버리지** | 해당 상권의 대체 접점이 있는가? | 반경 내 대체 매장 수·유동인구 커버 여부 |

3개 렌즈 모두 "닫아도 된다" → 폐점 유력 후보 / 1~2개 걸림 → 조건부 또는 유지 권고

## 2. 사용자 및 요구사항

### 사용자
| 구분 | 역할 | 주요 관심사 |
|---|---|---|
| 유통혁신팀 | 전국 포트폴리오 조정 결정 | 전국 현황 한눈에, 폐점 후보 목록, 시뮬레이션 결과 |
| 지역본부 | 권역 내 매장 효율 관리 | 내 권역 지도 보기, 특정 매장 상세 분석 |

### 핵심 요구사항
1. AI가 전국 매장에 대해 **폐점 위험도 스코어**를 자동 산출하고 순위를 보여줄 것
2. 특정 매장 선택 시 **3개 렌즈별 평가 + AI 종합 의견** 제공
3. "이 매장을 닫으면?" 시뮬레이션: M/S 변화·커버리지 공백·흡수 가능 매장 자동 계산
4. 경쟁사(KT/LGU+) 매장 위치를 지도에 오버레이하여 상권 공백 파악
5. 본사와 현장이 각자 편한 진입점(리스트 vs 지도)으로 접근 가능

## 4. 분석 방법론

### 4-0. 설계 철학

알고리즘이 "폐점하라"고 결론 내리지 않음. 각 분석 레이어가 서로 다른 관점의 신호를 제공하고, 사람(유통혁신팀·지역본부)이 신호를 종합해 최종 판단하는 구조.

매장 폐점에는 정해진 정답이 없음. 인구수 대비, LTV 관점, 경쟁 대응 관점 중 어느 하나로 환원되지 않으며, 전체 최적화보다 "여긴 줄여야 / 여긴 생겨야"의 지역별 신호 합으로 접근하는 것이 현실적.

가중 스코어링(weighted scoring) 방식은 직관적이지만 가중치 자체가 임의값이라는 근본적 한계가 있어 주된 방법론으로 채택하지 않음. 대신 각 레이어가 독립적인 관점을 제공하는 구조로 설계.

---

### 4-1. 레이어 1 — 지리적 밀집 구역 탐지 (DBSCAN 클러스터링)

리서치 ① 잠식 분석 → L1 DBSCAN + L3 Huff 모델로 반영

**목적**: "어디가 과밀인가"를 데이터가 스스로 찾게 함

DBSCAN(Density-Based Spatial Clustering)은 인근에 매장이 밀집된 구역을 자동으로 군집화. 사람이 기준을 정하지 않아도 데이터 분포에서 과밀 구역이 드러남.

```
입력: 전국 SKT 매장 위경도
출력: 클러스터 그룹 (밀집 구역) + 고립 매장 (단독 입지)

예시:
  클러스터 A: 수원시 팔달구 내 SKT 매장 5개 → 과밀 구역
  고립 매장:  강원 인제군 SKT 매장 1개 → 단독 입지, 폐점 불가
```

레이어가 주는 신호: "이 구역은 밀집되어 있어 중복 가능성이 있다" / 어느 매장을 닫을지는 레이어 2가 담당.

**파라미터**
- 반경(ε): 도심 0.8km / 수도권 1.5km / 지방 3.0km (상권 유형별 차등)
- 최소 매장 수(min_samples): 3개 이상이면 클러스터로 인정

---

### 4-2. 레이어 2 — 패턴 이탈 매장 탐지 (Isolation Forest)

리서치 ④ 채널 기능 분류 → L2 피처 설계에 반영 (채널 기능 유형 포함)

**목적**: "평균적인 매장 패턴에서 얼마나 벗어났는가"를 측정

Isolation Forest는 비지도학습 이상치 탐지 알고리즘. "폐점해야 한다"는 정답 레이블 없이, 전체 매장 중 패턴이 다른 매장을 탐지.

```
입력 피처 (매장별):
  - 월 순증 / 고객 평균 LTV / 월 운영비
  - 인근 SKT 매장 수 / 인근 경쟁사 매장 수
  - 상권 유동인구 / 채널 기능 유형 / M/S

출력: 매장별 anomaly score (높을수록 패턴 이탈)
```

이상치 = 폐점 후보가 아님. 이상치는 "주목해야 할 매장"이며, 이유는 다양.

| 이상치 유형 | 실제 의미 | 판단 방향 |
|---|---|---|
| 운영비 높음 + 순증 낮음 + 인근 SKT 多 | 잠식 받는 비효율 매장 | 폐점 검토 |
| 경쟁사 밀집 속 유일한 SKT 매장 | 전략적 요충지 | 유지 필수 |
| 출점 3개월 미만 | 데이터 미성숙 | 판단 보류 |
| 도서·산간 단독 입지 | 커버리지 유지 목적 | 유지 필수 |

레이어가 주는 신호: "이 매장은 평균과 다르다, 왜 다른지 들여다봐라" / 다르다고 해서 닫아야 한다는 결론은 아님.

---

### 4-3. 레이어 3 — 폐점 후 영향 시뮬레이션 (Huff 모델)

리서치 ① 잠식 분석 + 리서치 ② 커버리지·화이트존 → L3 Huff 모델로 반영

**목적**: "이 매장을 닫으면 실제로 어떻게 되는가" 검증

레이어 1·2에서 주목 매장이 식별되면, Huff 모델로 폐점 시 수요 이동을 확률적으로 예측.

```
Huff 모델 원리:
P(고객 i → 매장 j) = (매장규모_j / 거리_ij²) ÷ Σ(매장규모_k / 거리_ik²)

→ 특정 매장 제거 시 해당 매장 고객이 인근 SKT 매장으로 이동할 확률 계산
→ 경쟁사 매장으로 이탈할 확률도 추정
```

**시뮬레이션 출력**
1. **M/S 변화**: 폐점 후 해당 상권 SKT M/S 예상 변동폭
2. **흡수율**: 인근 SKT 매장이 수요를 흡수하는 비율
3. **이탈 위험**: 경쟁사로 넘어갈 것으로 추정되는 고객 수
4. **커버리지 공백**: 폐점 후 SKT 접점이 없어지는 유동인구 규모
5. **연간 비용 절감**: 운영비 × 12

레이어가 주는 신호: "닫았을 때 숫자로 어떤 일이 벌어지는가" / 닫아야 한다/말아야 한다는 결론은 아님.

---

### 4-4. 레이어 4 — 자연어 설명 (Claude API)

**목적**: 수치 결과를 의사결정자가 읽을 수 있는 언어로 변환

```
입력: 레이어 1~3의 분석 결과 (클러스터 위치, anomaly score, Huff 시뮬레이션 수치)
출력: "이 매장이 주목 대상이 된 이유"와 "닫을 경우 예상 시나리오" 자연어 요약
```

| 상황 | Claude 역할 |
|---|---|
| 매장 상세 패널 | "이 매장이 패턴 이탈로 탐지된 이유" 2~3줄 설명 |
| 시뮬레이션 결과 | "폐점 시 예상 시나리오 및 주의사항" 요약 |
| 권역 리포트 | "이 권역의 매장 구조에서 주목할 패턴" 인사이트 |

결론을 내리는 역할이 아니라 "왜 이 매장이 주목받는가"를 사람이 이해할 수 있도록 번역하는 역할.

## [참고] 외부 리서치 핵심 사항

상세 내용: `docs/research-retail-network-optimization.md`

### 업계 검증 방법론과 설계 반영 매핑

**① 잠식(Cannibalization) 분석** → L1 DBSCAN + L3 Huff 모델
인근 SKT 매장끼리 수요를 나눠 먹는 정도를 수치화해야 함. 매장 하나를 독립 평가하는 것은 불충분 — 인근 매장과의 관계를 반드시 함께 봐야 함.

**② 커버리지 공백 + 잠식 + 화이트존 3개 동시 분석** → L3 Huff 시뮬레이션 출력 + 지도 레이어
- 커버리지 공백: 우리가 없는 상권
- 잠식: 자사 매장끼리 겹치는 상권
- 화이트존: 경쟁사도 없고 우리도 없는 미개척 상권

지도에 3개 레이어로 시각화하면 "줄일 곳 / 늘릴 곳"이 직관적으로 드러남.

**③ SKT 자체 선례: CV센터 8곳 폐점 (2023)** → L2 Isolation Forest 피처 설계
상담 전용 단일 기능 매장을 전부 폐점하고 판매+상담 통합 직영대리점으로 전환. "어떤 기능을 하는 매장인가"가 폐점 판단의 중요 기준임을 SKT 스스로 실증 → 채널 기능 유형을 L2 피처로 포함.

**④ 글로벌 통신사 트렌드: 전체 매장 줄이되 기능별 양극화** → L2 이상치 해석 기준
미국 통신사 기준 후불 매장 소폭 감소, 선불(저관여) 매장 증가. 고가 서비스 채널은 통폐합, 저관여 채널은 확산하는 양극화 전략 → 이상치 유형 분류 시 채널 기능 기준 병행 적용.
